# Recipe Finder

Recipe Finder is a frontend web application built as part of the ALX Frontend Capstone Project.

The goal of this project is to allow users to search for recipes and view basic recipe details.  
This repository currently contains the initial project setup and skeleton structure.

## Tech Stack

- React (Vite)
- JavaScript
- TailwindCSS
- Zustand
- Axios

## Project Status

This project is currently in development.  
Part 3 focuses on setting up the project structure and starting the build process.

## Installation

To run the project locally:

```bash
npm install
npm run dev

```
